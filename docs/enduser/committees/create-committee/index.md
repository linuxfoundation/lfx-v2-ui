# Create a Committee

This article applies to users with **maintainer**, **board-member**, or **executive-director** roles. You can create new committees for your project directly from the LFX Self Serve portal.

## Before you begin

Make sure you have the correct project or foundation selected in the lens switcher. The new committee will be associated with the currently active project context.

## Steps

1. Sign in to [app.lfx.dev](https://app.lfx.dev).
2. Select **Groups** from the left navigation sidebar.
3. Select the option to create a new committee (route: `/groups/manage`).
4. Fill in the required fields:
   - **Committee name** — the display name for your committee
   - **Description** — a brief explanation of the committee's purpose and scope
5. Add initial members if needed.
6. Save the committee.

## After creation

The new committee appears in the committees dashboard. You can edit its details or add members at any time from the committee detail page.

## Permissions

Only users with maintainer, board-member, or executive-director personas can create committees. If you do not see the create option, your current role does not have the required access for this project context.
