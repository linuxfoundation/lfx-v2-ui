Use the Transactions section to review your purchase and billing history on the LFX platform.

## Steps

1. Sign in to [app.lfx.dev](https://app.lfx.dev).
2. Select **My Transactions** from the left navigation sidebar (route: `/me/transactions`).
3. Your transaction history appears in the main content area.
4. Select a transaction to view its details.

## What a transaction record shows

- Item name and description (for example, a training course or event registration)
- Transaction date
- Amount charged
- Select a transaction to view its details. Receipt or invoice download options, if available, appear in the transaction detail view.

## Filter transactions

Use the tabs at the top of the page to filter by transaction type: **All Transactions**, **Event Tickets**, **Training**, **Certifications**, **Bundles**, **Subscriptions**, **Individual Support**.

## Download receipts or invoices

Select a transaction to view its details. Receipt or invoice download options, if available, appear in the transaction detail view.

## Empty state

If you have no transactions, the page displays a message indicating no purchase history is available. Transactions are recorded only when you make a purchase through the LFX platform.
