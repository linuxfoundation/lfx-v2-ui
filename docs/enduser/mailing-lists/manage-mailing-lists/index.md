# Manage Mailing Lists

Use this page to subscribe to project mailing lists, manage your subscriptions, and (if you have the required role) update list settings.

## View mailing lists

1. Sign in to [app.lfx.dev](https://app.lfx.dev).
2. Select **Mailing Lists** from the left navigation sidebar.
3. The mailing lists dashboard shows all lists for your active project context.
4. Select a mailing list to open its detail view (route: `/mailing-lists/:id`).

## Subscribe to a mailing list

1. Open the mailing list you want to join.
2. Select the subscribe option on the list detail page.
3. After subscribing, check your email for a confirmation message from the mailing list.

## Unsubscribe from a mailing list

1. Open the mailing list you want to leave.
2. Select the unsubscribe option.
3. Confirm your choice when prompted.

## Manage list settings (maintainers and administrators)

1. Open the mailing list you want to manage.
2. Navigate to the manage or settings view (route: `/mailing-lists/:id/manage`).
3. Update the list name, description, or moderation settings as needed.
4. Save your changes.

## Switch project context

Mailing lists are scoped to a project or foundation. Use the lens switcher to change your active context and see lists for a different project.
