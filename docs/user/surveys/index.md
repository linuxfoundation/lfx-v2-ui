# Surveys

Creating and managing surveys requires a **maintainer**, **board-member**, or **executive-director** persona. Contributors can respond to surveys they receive but cannot create them.

The Surveys section lets you create surveys for your project community, collect responses, and analyze feedback including Net Promoter Score (NPS) data. Surveys are a tool for gathering structured input from contributors, maintainers, and stakeholders.

## What you can do

- Create new surveys for your project
- Edit existing surveys
- Collect and review responses
- View NPS analytics for your project
- Share surveys with respondents via a survey link

## Navigation

Go to **app.lfx.dev** and select **Surveys** from the left navigation sidebar. The surveys dashboard (route: `/surveys`) shows all surveys for your active project context. Use the tabs to filter: **All**, **Open**, **Closed**.

## Key concepts

- **Net Promoter Score (NPS)**: A metric measuring how likely respondents are to recommend your project on a scale of 0–10
- **Survey**: A set of questions sent to a defined audience for structured feedback collection
