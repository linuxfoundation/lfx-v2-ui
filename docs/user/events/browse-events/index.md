# Browse Events

Use the Events section to find Linux Foundation events relevant to your projects and interests.

## Steps

1. Sign in to [app.lfx.dev](https://app.lfx.dev).
2. Select **Events** from the left navigation sidebar.
3. Browse the list of upcoming and past events.
4. Select an event to view its full details.

## What an event listing shows

The events table shows the following columns:

- **Event Name**
- **Foundation** — the foundation associated with the event
- **Date**
- **Location**
- **Role**
- **Status**

Registration is handled externally. The Events page displays events you are already registered for.

## Filter events

Filter by **All Foundations**, **All Roles**, **All Statuses** using the dropdown menus. Use the tabs to switch between **Upcoming**, **Past**, **Visa Letters**, and **Travel Funding**.

## Register for an event

LFX Self Serve shows your registered events. To register for new events, visit the event's external registration page.

## Empty state

If no events appear, your current project context may not have any scheduled events. Try switching your lens to see events for a different scope, or check back later.
