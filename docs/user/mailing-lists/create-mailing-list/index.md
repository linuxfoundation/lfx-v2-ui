# Create a Mailing List

This article applies to users with **maintainer**, **board-member**, or **executive-director** personas. You can create new mailing lists for your project to support community communications.

## Before you begin

Confirm you have the correct project selected in the lens switcher. The new mailing list will be associated with the currently active project context.

## Steps

1. Sign in to [app.lfx.dev](https://app.lfx.dev).
2. Select **Mailing Lists** from the left navigation sidebar.
3. Select the option to create a new mailing list (route: `/mailing-lists/manage`).
4. Fill in the required fields:
   - **List name** — the display name and identifier for the mailing list
   - **Description** — a brief explanation of the list's purpose
5. Save the mailing list.

## After creation

The new mailing list appears in the mailing lists dashboard. Community members can subscribe to it from the list detail page. You can edit the list settings at any time.

## Permissions

Only users with maintainer, board-member, or executive-director personas can create mailing lists. If you do not see the create option, contact your project administrator.
