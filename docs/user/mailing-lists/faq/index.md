## Who can subscribe to a mailing list?

All authenticated LFX users can subscribe to project mailing lists. Some lists may be restricted to project members or have moderation enabled.

## Who can create a mailing list?

Users with **maintainer**, **board-member**, or **executive-director** personas can create mailing lists. Contributors can subscribe to existing lists but cannot create new ones.

## Why don't I see any mailing lists?

Confirm your active project context in the lens switcher. Mailing lists are scoped to a specific project or foundation. If the correct project is selected and you still see no lists, the project may not have created any mailing lists yet.

## How do I unsubscribe from a mailing list?

Open the mailing list from the **Mailing Lists** section and select the unsubscribe option on the list detail page.

## What happens to a mailing list when the project is archived?

If a project is archived, contact LFX support for information about accessing its mailing list history.

## How do I change the mailing list name or description?

Open the mailing list detail page and navigate to the manage or settings view. Update the fields and save your changes. This requires a maintainer or higher persona.

## Who do I contact if a mailing list is set up incorrectly?

Contact LFX support through the Help Center. Include the list name and the project it belongs to.
