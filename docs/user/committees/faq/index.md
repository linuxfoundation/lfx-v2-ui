# Committees FAQ

## Who can create a committee?

Users with a **maintainer**, **board-member**, or **executive-director** persona can create committees. Contributors can view committee information but cannot create or manage committees.

## Why can't I see any committees?

Make sure you have the correct project or foundation selected in the lens switcher. Committees are scoped to a specific project context. If you switch to the **Foundation** lens, you see foundation-level committees; the **Project** lens shows project-level committees.

## How do I add members to a committee?

Open the group from the **My Groups** page, navigate to the members section, and use the add member option to search and add by name or email.

## Can I have the same person in multiple committees?

Yes, a single user can be a member of multiple committees within the same project or across different projects.

## How do I change the committee name or description?

Open the committee detail page and select the edit option. Update the fields and save your changes.

## What happens to a committee's meetings and votes when the committee is deleted?

Contact your project administrator before deleting a committee, as this may affect associated meetings, votes, and documents.

## Who do I contact if I need help with committees?

Contact LFX support through the Help Center. Include your project name and the committee name in your request.
